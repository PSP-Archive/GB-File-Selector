GB File Selector

選んだファイルをメモリースティックの/PSP/GAME/RIN/ROM.GB にコピーします（謎）
本家がファイル選択を実装するまでのつなぎとも思いますが一応。
そんな感じででっちあげただけなので、画面とか素っ気無いですけど気にしない。

/PSP/GAME/ 以下に適当な名前のフォルダを作り、その中にEBOOT.PBPをコピーしてください。

他にももういっこポカミス修正

0.03

なにやらファイルセレクタ部や日本語フォント関連で参照されまくっているので
（いーかげんに書いたソースなんだけどなー）
とりあえず更新しておきます。HOMEボタン終了と、フォント第二水準対応。





※注意※
あらかじめ
/PSP/GAME/RIN/ROM.GB
に何かファイルを置いておいてください。（ファイルの新規作成はできません。なぜか上書き更新しかできない仕様）
